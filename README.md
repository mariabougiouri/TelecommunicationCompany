# Telecommunication Company
## The functions of this project are:
### For Company:
  - Add new company
  - Modify company
  - Company search
  - Delete company
### For Plan:
  - Add new plan (mobile or landline)
  - Plan search
  - Modify plan
  - Delete plan
### For Client:
  - Add client
  - Client search
  - Modify of client information
  - Delete client 
### For Contract:
  - Add contract
  - Contract search
  - Cancellation of a contract
